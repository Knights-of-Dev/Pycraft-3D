# Pycraft 3D
Have you wanted a Minecraft copy that was really really BAD? Like, worse than the first version? Try ***Pycraft 3D***! It's so fun!

## Software Requirements
You need the following things:
- python 3.12+
- ursina module (do ```pip install ursina``` or ```python -m pip install --user ursina```)
- a **SOUL**
- the unzipped ZIP file from this repo's latest release

## Controls
### Movement
Use WASD to move around.
### Looking
Use the mouse to look and interact with the environment.
