# Pycraft 3D
Have you wanted a Minecraft copy that was really really BAD? Like, worse than the first version? Try ***Pycraft 3D***! It's so fun!

## Controls
### Movement
Use WASD to move around.
### Looking
Use the mouse to look and interact with the environment.

- A way to screenshot what you want!
